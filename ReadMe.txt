管理者として実行

■ これは何?
コマンド(プログラム)を管理者として実行します。

■ 使い方
sudo コマンド 引数...

■ セットの仕方
install.ps1 を実行してください

次に PowerShell プロンプト開くと sudo コマンドが使えるようになっています。

■ Uninstall 方法
uninstall.ps1 を実行して下さい
(問い合わせが来たら Enter)

■ 動作確認環境
PowerShell 5.1
Windows 環境専用です。
